# 👔 Sistema de Ventas POS - Macho's

Sistema de gestión de inventario y ventas desarrollado en PHP Puro (Sin frameworks) implementando el patrón de arquitectura MVC.

## 🚀 Características
- Gestión de Inventario (Tallas, Colores, Stock).
- Punto de Venta (POS) rápido.
- Reportes en PDF y Gráficos estadísticos.
- Control de Caja (Apertura, Cierre y Arqueo).
- Gestión de Usuarios y Roles.

## 🛠️ Tecnologías
- PHP 8.x
- MySQL / MariaDB
- Bootstrap 5
- JavaScript (Chart.js)
- HTML5 / CSS3

## 📦 Instalación
1. Clona este repositorio.
2. Importa el archivo `database/backup.sql` en tu gestor de base de datos (HeidiSQL/phpMyAdmin).
3. Configura la conexión en `app/config/Database.php`.
4. ¡Listo! Accede desde `localhost/carpeta_proyecto`.
